7. About This Tool

This project recreates a core planning function of the historic CALIPR preservation survey workflow: estimating a statistically valid sample and converting that sample into randomized physical stack locations.

  • Useful for large-scale condition assessment when item-by-item review is impractical.
  • Designed for stacks modeled by range, side, bay, shelf, and item.
  • Built to support both statistical rigor and practical walking order in the stacks.
Practical use case

Generate an initial survey batch, work the entire list in order, and use a top-up batch only if empty or unusable locations reduce the final usable sample.

8. Methods & Notes

The sample size estimate uses a conservative proportion (p = 0.5) with finite population correction. Planned draws are then inflated based on occupancy to account for likely misses in physical stack sampling.

  • Target sample (n): statistically adjusted survey size.
  • Planned draws: total randomized locations generated after occupancy adjustment.
  • Walking order: a sorted version of the same sampled rows for efficient movement.
Important sampling caution

To avoid introducing bias, do not stop partway through a generated batch. Attempt every row, then generate a separate top-up batch if needed.
